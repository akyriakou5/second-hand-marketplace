## About The Project

This project is a web application that allows users to buy and sell second-hand products online.  
The system provides a simple marketplace where sellers can upload products and visitors can browse available products.


The platform includes a seller dashboard where sellers can manage their products by creating, editing, and deleting listings.
Also, the system allows sellers to mark their products as "Available" or "Sold".

Visitors can explore all available products, filter them by category, and view detailed product information including images, price, description, and seller contact information.


The main goal of this project is to promote the reuse of products and encourage people to buy second-hand items, helping reduce waste and save money.

## Getting Started

This section explains how to set up and run the project on a local computer. 
The following steps describe the software required and the instructions needed
to install and run the application using XAMPP and a web browser.

### Prerequisites

Before running the project, the following software must be installed:
* XAMPP
* Web browser
  
### Installation

Follow these steps to run the project locally:
 
1. Install XAMPP (8.2.12) : https://www.apachefriends.org/
2. Open the XAMPP Control Panel and start:
   - Apache
   - MySQL
3. Copy the project folder (DoubleProject) into: 
c:\xampp\htdocs\
4. Open the XAMPP Control Panel.
5. Click on the "Admin" button next to MySQL to open phpMyAdmin in your browser.
6. Create a new database named:
   secondhand_products
7. Import the provided database file:
   secondhand_products.sql
8. Open your browser and go to:
   http://localhost/DoubleProject/

The application should now run locally.



### Important Folder Setup

If you download the project as a ZIP file from GitHub, the folder name will usually be:

DoubleProject-main

You must rename this folder to:

DoubleProject

Make sure that the project structure is correct:

✔ Correct:
C:\xampp\htdocs\DoubleProject\ (project files here)

❌ Incorrect:
C:\xampp\htdocs\DoubleProject\DoubleProject\ (nested folder)

The project files (index.php, assets, etc.) must be directly inside the "DoubleProject" folder.

Otherwise, the application may not run correctly.

After that, open your browser and go to:
http://localhost/DoubleProject/

## Usage

The application provides two main user roles: visitors and sellers.

Sellers can:
* Register an account
* Login to their dashboard
* Create new products
* Upload images
* Edit products
* Mark products as Sold or Available
* Delete products

Visitors can:
* Browse all products
* Filter products by category
* View product details
* Contact the seller using their email address
* Visit a seller's profile to see all their products (available and sold)
